Você foi contratado para desenvolver uma funcionalidade de consulta de endereço por CEP utilizando a API pública do ViaCEP (https://viacep.com.br). Seu objetivo é criar um Sistema com JavaScript, HTML e CSS que: • Permita o usuário digitar um CEP. • Busque os dados do endereço correspondente utilizando a API. • Exiba o resultado na tela de forma clara. Requisitos obrigatórios: 1. A página deve conter um campo de entrada para o CEP e um botão para realizar a busca. 2. Ao clicar no botão, deve ser feita uma requisição fetch para a API https://viacep.com.br/ws/{cep}/json/. 3. Os dados retornados devem ser exibidos no navegador: logradouro, bairro, localidade e UF. 4. O sistema deve tratar CEPs inválidos (exemplo: menos de 8 dígitos ou erro da API).
